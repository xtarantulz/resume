# Персональне портфоліо

## Про проект

Цей репозиторій містить моє **персональне портфоліо**, створене для демонстрації мого досвіду, навичок та проектів як **Senior PHP / Fullstack Developer**.  
Я вирішив розмістити портфоліо на GitHub, щоб:

- Мати версіоновану копію проекту.
- Продемонструвати чисту структуру та організацію коду на HTML/CSS/JS.
- Показати підтримку кількох мов (українська, російська, англійська) через динамічну підвантаження контенту.
- Легко ділитися портфоліо з роботодавцями або клієнтами.
